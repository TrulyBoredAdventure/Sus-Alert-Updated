# Release Notes - 1.3.0

## Movable next-special overlay

- Added an in-game next-special display with one compact pictogram, the special name, and remaining seconds.
- Added urgency colours as the countdown approaches zero.
- Added small, medium, and large sizes.
- Added Move, Preview, Reset, and Cancel controls in settings.
- Saved the overlay position relative to the RuneScape client.
- Integrated countdown timing with encounter start and reset, core pauses, manual timer changes, and middle-fungus resynchronisation.
- Kept all code and assets local to the standalone repository.
- Added automated timing, browser-drawing, placement, and settings checks.

## Upgrade requirement

Replace the complete repository contents, allow GitHub Pages to redeploy, remove the previous app entry from Alt1, and install it again using the main `appconfig.json` link.
