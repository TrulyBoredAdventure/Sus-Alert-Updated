SusAlert Updated

SusAlert starts and resets automatically when it detects the Croesus encounter. The route tracker is read-only and requires the player to confirm each completed route action.

RuneScape setup

  • Keep the boss timer enabled and unobstructed.
  • Use 100 percent interface scaling.
  • Keep game messages enabled.
  • Enable local chat timestamps for more reliable chat reading.
  • Select the correct chat box in SusAlert settings.
  • Use a chat text size of at least 12.
  • Use 0 percent interface transparency when statue indicators are enabled.

Next-special overlay

The in-game overlay shows a compact pictogram, the next special, and the remaining seconds. Open settings, select Move, position the pointer in RuneScape, then return to settings and select Set.

Route tracker

Select a party size, team role, and rotten fungus duty before the encounter. Use the main action button only after completing that action in game. Counter controls are available for corrections.

Read-only operation

The app reads permitted screen and game-state information, displays guidance, and stores settings locally. During overlay placement it observes the pointer position, but it does not click, type, move the mouse, select targets, or perform RuneScape actions.
